# Презентация: Атаке-бий (1737-1787)

## Описание
Историческая презентация о великом кыргызском полководце и дипломате Атаке-бие. Презентация содержит 10 слайдов с подробной информацией о его жизни, военных кампаниях и дипломатической деятельности.

## Как открыть презентацию

1. **Откройте файл в браузере:**
   - Дважды кликните на файл `atake-biy-presentation.html`
   - Или откройте его в любом современном браузере (Chrome, Firefox, Safari, Edge)

2. **Навигация по слайдам:**
   - Используйте кнопки "Назад" и "Вперед" внизу экрана
   - Или используйте клавиши ← и → на клавиатуре
   - Счетчик показывает текущий слайд (например: 1 / 10)

## Как конвертировать в PPTX

### Способ 1: Через Google Slides (Рекомендуется)

1. **Откройте презентацию в браузере** (файл `atake-biy-presentation.html`)

2. **Сделайте скриншоты каждого слайда:**
   - macOS: `Cmd + Shift + 4`, затем пробел, кликните на окно браузера
   - Переключайтесь между слайдами стрелками и делайте скриншоты всех 10 слайдов

3. **Создайте новую презентацию в Google Slides:**
   - Перейдите на https://slides.google.com
   - Создайте новую презентацию
   - Установите размер слайда: Файл → Настройка страницы → 16:9

4. **Вставьте скриншоты:**
   - Для каждого слайда: вставьте изображение во весь слайд
   - Удалите текстовые блоки по умолчанию

5. **Скачайте в PPTX:**
   - Файл → Скачать → Microsoft PowerPoint (.pptx)

### Способ 2: Онлайн-конвертеры

1. **CloudConvert** (https://cloudconvert.com/html-to-pptx)
   - Загрузите HTML файл
   - Выберите формат PPTX
   - Нажмите "Convert"
   - Скачайте результат

2. **Zamzar** (https://www.zamzar.com/convert/html-to-pptx/)
   - Загрузите HTML файл
   - Выберите PPTX как выходной формат
   - Конвертируйте и скачайте

### Способ 3: Через MS PowerPoint (Windows)

1. Откройте PowerPoint
2. Создайте пустую презентацию размером 16:9
3. Откройте HTML файл в браузере в полноэкранном режиме (F11)
4. Используйте Snipping Tool или PrintScreen для каждого слайда
5. Вставьте скриншоты в PowerPoint как фоновые изображения

### Способ 4: Печать в PDF, затем PDF в PPTX

1. **Создайте PDF из HTML:**
   - Откройте `atake-biy-presentation.html` в браузере
   - Для каждого слайда: Cmd+P (macOS) или Ctrl+P (Windows)
   - Выберите "Сохранить как PDF"
   - Повторите для всех 10 слайдов

2. **Конвертируйте PDF в PPTX:**
   - Используйте Adobe Acrobat или онлайн-сервисы:
     - https://www.adobe.com/acrobat/online/pdf-to-ppt.html
     - https://smallpdf.com/pdf-to-ppt
     - https://www.ilovepdf.com/pdf_to_powerpoint
